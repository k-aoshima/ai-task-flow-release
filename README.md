## 🚀Chrome拡張機能クイックスタート
1. [releases](https://github.com/k-aoshima/ai-task-flow-release/releases)からai-task-flow.zipをダウンロードし、解凍
2. google chromeの設定から「拡張機能」> 「拡張機能を管理」
3. 拡張機能を管理ページの右上に「ディベロッパーモード」のトグルがあるのでONにする
4. 画面左上に「パッケージ化されていない拡張機能を読み込む」があるのでクリックし、1で解凍したフォルダを選択する
5. 拡張機能が追加されるので、AI TaskFlowを開き画面右上の設定ボタンをクリックする
6. https://aistudio.google.com/ でAPIキーを作成　※APIキーの取得方法は[こちら](https://ai.google.dev/gemini-api/docs/api-key?hl=ja)を確認してください。無料で発行可能です。
7. AI TaskFlowに戻り、取得したGemini APIキーを設定にいれてください

## 機能

### コア機能
- **AIタスク分解**: 大きなタスクを実行可能なサブタスクに自動分解
- **優先度スコアリング**: 緊急度・重要度・作業環境に基づく自動スコアリング
- **タスク編集**: タスク名、親タスク名、時間、環境をクリックで直接編集
- **グループ管理**: 親タスクでサブタスクをグループ化、展開/折りたたみ可能
- **ドラッグ&ドロップ**: タスクやグループを自由に並び替え、リスト間で移動
- **実行中リスト**: 今すぐ実行するタスクを別リストで管理
- **データ永続化**: localStorageによるデータ保存

### 🧠 AIによるコンテキスト認識

#### 仕組み
1. **コンテキストの取得**: Chrome拡張機能が現在のタブのURL、タイトル、ドメインを自動取得
2. **キーワードマッチング**: タスクに紐づけられたキーワードと現在のタブ情報を照合
3. **動的ブースト**: キーワード一致度に基づいて、タスクのスコアを最大2.5倍までブースト
4. **リアルタイム更新**: タブを切り替えると、タスクリストが自動的に並び替えられる
5. **ドメインパターン**: 特定のサイト（GitHub、Gmail等）とタスクを関連付け

#### 実例
**例**: Gmailで「[クライアントA] 企画書修正依頼」というメールを開くと、関連キーワード（Gmail、クライアントA、企画書など）を含むタスクが自動的にリストのトップに表示され、「✨ AIおすすめ」バッジが付きます。

### タイマー機能
- **保留タイマー**: タスクを一時保留し、指定時間後に通知
- **プリセット時間**: 5分、10分、30分、1時間、カスタム
- **残り時間表示**: リアルタイムでカウントダウン表示

## 使用方法

### 初期セットアップ

1. 拡張機能のアイコンをクリックしてポップアップを開く
2. 設定画面（⚙️アイコン）でGemini APIキーを入力
3. ドメインパターンを設定して、特定のサイトとタスクを関連付け

### タスクの作成と管理

#### タスクの追加
1. メイン画面のフォームにタスク名を入力
2. 「AIでタスクを分解」をクリック
3. AIが自動的にタスクを分解し、サブタスクを生成

#### タスクの編集
- **タスク名の変更**: タスク名をクリックして直接編集（Enter で確定、Esc でキャンセル）
- **親タスク名の変更**: グループヘッダーの親タスク名をクリックして編集（すべての子タスクの親タスク名も自動更新）
- **時間の変更**: 「⏱ XX min」をクリックして編集
- **環境の変更**: 「📍 環境名」をクリックして選択

#### タスクの操作
- **完了**: ✓ ボタンをクリック
- **削除**: 🗑️ ボタンをクリック
- **保留**: ⏸️ ボタンをクリックして保留時間を設定
- **再分解**: 🧪 ボタンをクリックしてタスクを再度AIで分解

### グループ管理

#### グループの展開/折りたたみ
- グループヘッダーの矢印アイコンをクリック

#### グループ内タスクの完了
1. すべての子タスクにチェックを入れる
2. グループヘッダーの ✓ ボタンが有効化される
3. クリックしてグループ全体を完了

### ドラッグ&ドロップ

#### タスクの並び替え
- タスクをドラッグして好きな位置にドロップ
- グループ全体もドラッグ可能

#### 実行中リストへの移動
- タスクまたはグループを「🚀 実行中のタスク」リストにドラッグ
- 実行中リストに追加されたタスクは優先的に表示

#### 完了リストへの移動
- タスクを「完了リスト」エリアにドラッグ
- グループの場合は、すべての子タスクにチェックが必要

### コンテキスト認識

#### 自動優先度調整
1. タスクにキーワードを設定（AI分解時に自動設定）
2. 現在のタブ情報（URL、タイトル、ドメイン）と照合
3. 関連性の高いタスクに「✨ AIおすすめ」バッジが表示
4. タブを切り替えると自動的に優先度が再計算

#### ドメインパターンの設定
1. 設定画面で「ドメインパターン管理」を開く
2. パターン名、URLパターン、関連キーワードを設定
3. 例: `github.com` → キーワード「コード、レビュー、プルリク」

### タイマー機能

1. タスクの保留ボタン（⏸️）をクリック
2. 保留時間を選択（5分、10分、30分、1時間、カスタム）
3. タイマーが開始され、残り時間が表示
4. タイマー終了時に通知が表示
